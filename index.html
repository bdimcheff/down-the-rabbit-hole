Act IV: Diagnosis

no close :(

wireshark screenshot

closed by the client

Shotgun Debugging

I stuck breakpoints everywhere that said close

and it worked!

  1. old websocket is swapped out for new on reconnect
  2. old websocket's heartbeat is not properly cleaned up
  3. old websocket's heartbeat calls disconnect
  4. new websocket executes reconnect logic

But who cares...

Act V: The Fix

-    this.isOpen = false;
-    this.socket.onClose();
-    this.onDisconnect();
+    /* only call close if transport hasn't been swapped out already */
+    if (this.socket.transport === this) {
+      this.isOpen = false;
+      this.socket.onClose();
+      this.onDisconnect();
+    }
